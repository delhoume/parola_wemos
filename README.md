# wemos_parola

A ticker written with Parola, for 2 or 3 4x8x8 led modules.
Works with esp8266

- Time via NTP
- Bitcoin price
- OpenWeatherMap local weather
- Stock quotes
- Instagram followers
- Adafruit.IO feed value
- Custom text via web interface

2 parola zones:
- 3 modules for time
- rest for scrolling contents
